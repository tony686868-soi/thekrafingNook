This personal tool DOES NOT:

2. Data Usage

The tool only accesses the shop owner's own Etsy data through official API endpoints for personal inventory management. All data processing happens locally and temporarily during active use.

3. Third-Party Services

The only external service used is the official Etsy API v3, accessed through secure OAuth 2.0 authentication. No other third-party services are integrated.

4. Data Security

- All API communications use HTTPS encryption
- OAuth tokens are stored securely and locally
- No data is transmitted to external servers
- Access is limited to the authenticated shop owner only
